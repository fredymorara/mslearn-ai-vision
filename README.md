# Azure AI Computer Vision Projects

Welcome to my Azure AI Computer Vision repository! This repository is a collection of projects demonstrating my hands-on experience developing advanced computer vision and generative AI solutions using **Microsoft Azure AI Services (Foundry)** and **Azure OpenAI**.

These labs showcase my ability to integrate multi-modal AI models, implement robust passwordless authentication, handle advanced API patterns (like asynchronous polling), and adapt to architectural challenges (such as regional model availability).

---

## 🚀 Projects Showcased

### 1. Generative AI Multi-Modal Chat (`gen-ai-vision`)
Built an interactive Python chat application capable of analyzing images and text simultaneously. 
* **Capabilities:** Integrated Azure OpenAI's latest models (e.g., `gpt-5.2`) to answer complex queries about images.
* **Techniques Used:** Successfully handled both URL-based image inputs and local image uploads by encoding files to `base64` data URLs. Structured API requests with the newest developer/user role paradigms.

### 2. Custom Image Generation Pipeline (`image-client`)
Adapted a standard image generation pipeline to utilize a partner model via Azure AI Serverless endpoints.
* **Problem Solving & Adaptability:** When the standard DALL-E model was unavailable in the current Azure region, I pivoted to use the **FLUX.2-pro** model by Black Forest Labs.
* **Techniques Used:** Because this specific partner model required a raw REST API call rather than the native OpenAI SDK, I refactored the application to use the `requests` library. I securely injected bearer tokens generated by `DefaultAzureCredential` (using the `https://ai.azure.com/.default` scope) directly into the HTTP headers to fetch and decode the resulting `b64_json` image.

### 3. Video Generation & Remixing (`video-generation`)
Developed a comprehensive script for generating and manipulating video content using advanced generative models (like Sora).
* **Capabilities:** Implemented text-to-video, image-to-video, and video-remixing pipelines.
* **Techniques Used:** Built robust, asynchronous state-polling loops (`poll_video_status`) to continually check the Azure endpoint and handle long-running model execution safely before downloading the final MP4.

### 4. Intelligent Content Understanding (`content-understanding`)
Created a robust script to extract metadata, descriptions, and tags from local images using the Azure Content Understanding SDK.
* **Capabilities:** Leveraged the `ContentUnderstandingClient` to parse physical files.
* **Techniques Used:** Implemented specific polling logic (`begin_analyze`) and complex JSON traversal to cleanly output structured tags and metadata.

## 🔐 Security & Best Practices
Across all projects in this monorepo, I prioritized enterprise-grade security and environment hygiene:
* **Passwordless Authentication:** Implemented `DefaultAzureCredential` from the `azure.identity` library across all scripts to ensure secure, token-based authentication without hardcoding API keys.
* **Environment Isolation:** Relied heavily on `python-dotenv` for configuration, ensuring that endpoint URIs and deployment names remain out of the source code.
* **Git Hygiene:** Maintained a strict `.gitignore` globally across the monorepo to ensure that virtual environments (`labvenv`) and `.env` files are never tracked or committed to version control.
